怎么判断质数

判断一个数是不是质数（质数=只能被 1 和它本身整除的自然数）的方法有快有慢，取决于这个数有多大：

1. 小范围（几十万以内）

直接用试除法，并加上几个优化：

  1. 小于 2 → 不是质数；2 和 3 → 质数。

  2. 如果能被 2 或 3 整除 → 不是质数。

  3. 从 5 开始试除到 √n，只检查形如 6k ± 1 的数：

    import math
def is_prime(n):
    if n < 2:
        return False
    if n in (2, 3):
        return True
    if n % 2 == 0 or n % 3 == 0:
        return False
    limit = int(math.isqrt(n))
    i = 5
    while i <= limit:
        if n % i == 0 or n % (i + 2) == 0:
            return False
        i += 6
    return True

    这样能减少大约 2/3 的除法次数。

马尔科夫链的应用

解释马尔可夫链

马尔可夫链 (Markov Chain) 听起来像是一个复杂的数学术语，但它的核心概念非常简单，甚至可以说是“直觉”的。

简而言之，马尔可夫链是一个描述状态随时间变化的数学模型。

它的核心规则（也是它最著名的特征）叫做无记忆性（Memorylessness），即：

“下一步会发生什么，只取决于现在的状态，而与过去发生了什么无关。”
